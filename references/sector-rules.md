# 行业对标选择规则

用于在研究报告中构建估值对标模块。

## 物业数字化 / 物业管理

- **目标场景**：千丁、物业数字化
- **核心对标**：
  - 明源云 (00909.HK) - 行业标杆
  - 万物云 (02602.HK) - 物业科技龙头
  - 碧桂园服务 (06098.HK) - 传统大厂转型
  - Xylem (XYL) - 数字化水处理（Ecolab 逻辑）
  - Salesforce (CRM) - SaaS 溢价参照

## 工业 SaaS / 设施管理

- **目标场景**：能耗管理、工业运维
- **核心对标**：
  - Ecolab (ECL) - 服务+订阅模式金标准
  - Honeywell (HON) - 工业自动化
  - Johnson Controls (JCI) - 楼宇自控
  - Schneider Electric (SU.PA) - 电力与能源管理

## 文创 / IP 品牌

- **目标场景**：品牌化、粉丝经济
- **核心对标**：
  - 泡泡玛特 (09992.HK) - 潮流 IP
  - 石头科技 (688169.SH) - 小米生态链
  - 周大福 (01929.HK) - 贵金属工艺
  - 名创优品 (09896.HK) - 质价比零售
