# UI 模板（Web Report）

## 龙湖品牌色

- 深蓝 `#023894` - 标题、导航
- 橙色 `#E97118` - 强调、按钮
- 青色 `#01B2B3` - 辅助色

## 核心模块

### 1. 财务概览
3-5 年趋势图表（营收、利润、现金流）

### 2. 业务分部（Master-Detail）
- 左侧：业务板块列表
- 右侧：选中板块的深度分析
- 切换指标：营收 / 营业利润

### 3. 估值对标
- 可比公司选择（从 sector-rules.md 读取）
- 实际 vs Forward 切换
- 动态计算中位数/均值
- Target 相对 Peer 的溢价百分比

### 4. 千丁启示
- 认知迁移
- 风险对照
- 决策模式参考

### 5. 认知层可视化
- 思维模式网络图（D3.js force-directed）
- 决策启发式卡片（可交互）
- 验证状态标色（绿✅/黄⚠️/红❌）

## 技术栈

React 18 + Tailwind + Chart.js + D3.js，单 HTML 文件。
