# 财务数据采集清单

## 核心指标

| 指标 | 说明 | 来源 |
|------|------|------|
| 历年营收（10年+） | 规模与增长趋势 | 10-K / StockAnalysis / Macrotrends |
| 净利润 | 盈利能力 | 同上 |
| 营业利润 | 运营效率 | 同上 |
| 业务分部收入 | 各板块收入及占比 | 10-K 年报 |
| 经常性收入占比 | 商业模式质量 | MD&A / 投资者日材料 |
| 毛利率 | 定价能力 | 10-K |
| 营业利润率 | 运营效率 | 10-K |
| 自由现金流 | 现金创造能力 | 10-K |
| ROIC | 资本配置效率 | 10-K |
| 员工人数 | 组织规模 | 10-K |

## 数据验证

- 交叉验证：关键数字至少两个来源
- 时间一致性：财年口径统一（FY vs Calendar Year）
- 缺口标记：NA 表示不可得，不编造
