# 思维模型提取方法论

## 核心目标

提取目标公司的"认知操作系统"（思维模型、决策启发式、价值观），而非罗列事实数据。

## 五阶段流程

```
1. 身份确认 → 2. 六路采集 → 3. 认知提取 → 4. 双重产出 → 5. 归档交付
```

### 阶段 1：身份确认
确认公司唯一标识（全称、股票代码、上市地、基础背景）。

### 阶段 2：六路并行采集
同时从 6 个维度收集信息：
- **A 财务**：数字背后的态度与解读框架
- **B 业务**：价值创造逻辑与护城河认知
- **C 战略**：重大决策时间线与决策模式
- **D 领导思维**：管理层思维框架与决策原则
- **E 批判视角**：反对意见与风险盲区
- **F 文化符号**：组织文化与非正式规则

每个研究流输出两个文件：
- `raw.md`：原始素材与 URL
- `cognitive.md`：提取的思维模型（已标注验证状态）

### 阶段 3：认知提取与验证
1.  从各研究流提取思维模型、决策启发式、价值观
2.  应用三重验证（跨域/预测/排他）
3.  保留矛盾，不强行统一

### 阶段 4：双重产出
- **company-skill.md**：可直接 `\load` 的 AI Skill
- **Web Report**：含认知可视化的 HTML 报告

### 阶段 5：归档交付
选项：存入飞书知识库、更新千丁上下文、生成 Obsidian 卡片

## 六路研究流要点

### A. 财务数据 → 数字直觉
**提取**：该公司如何定义"健康"的财务指标？业绩不达标时的解释框架？
**信号**：营收增长 vs 利润率权衡、现金流优先级、非线性增长容忍度

### B. 业务模式 → 价值创造逻辑
**提取**：他们认为自己在解决什么本质问题？不可替代性是什么？
**信号**：价值主张的独特诠释、生态绑定、问题重新定义

### C. 战略时间线 → 决策启发式
**提取**：重大决策的触发条件与重复模式
**格式**："当 [情境] 出现时，我们总是 [行动]"
**信号**：危机驱动转型、小步快跑扩张、收购整合优先

### D. 领导思维 → 认知框架（最重要）
**提取**：CEO/创始人如何思考问题？引用哪些思想家的理论？
**信号**：第一性原理、逆向工作、边际思维、系统思维

### E. 批判视角 → 风险盲区
**提取**：批评者看到的弱点、未被有效反驳的指控
**信号**：规模不经济、文化稀释、过度依赖灵魂人物

### F. 文化符号 → 非正式规则
**提取**：组织不成文的潜规则、禁忌、英雄叙事
**信号**：内部反复讲述的故事、晋升样本特征、公开批评的行为

## 验证标准

每条思维模型必须满足至少两项：
- **跨域出现**：出现在 ≥2 个独立研究流
- **预测能力**：能解释 ≥1 个已知历史决策（吻合度 ≥70%）
- **排他性**：不是行业通用常识，而是该公司独特认知

## 质量控制

- 每条模型：来源 + 验证状态 + 置信度 三要素齐全
- 矛盾观点明确记录，不回避
- 对未知问题主动标注不确定性
- 所有模型可追溯至原始来源
